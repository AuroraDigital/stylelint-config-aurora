
# stylelint-config-aurora


Stylelint, but with opinionated changes.



## Changed rules

None yet


## Usage

**Install:**
```
yarn add --dev stylelint-config-aurora
```
or
```
npm install --save-dev stylelint-config-aurora
```

**Edit `package.json`**:
```
{
  // ...
  "stylelint": {
    "extends": "stylelint-config-aurora
  }
}
```
